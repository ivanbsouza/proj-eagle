# 🦅 Projeto Águia

Projeto full-stack com React frontend e Node.js backend com PostgreSQL.

## 🚀 Tecnologias

- **Frontend**: React, JavaScript, HTML, CSS
- **Backend**: Node.js, Express.js, PostgreSQL
- **Outras**: CORS, Dotenv, PG

## 📋 Pré-requisitos

- Node.js 14+
- PostgreSQL 12+
- npm ou yarn
